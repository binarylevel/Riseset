Riseset
===============

Tracking weather with [http://forecast.io](http://forecast.io)

[![Build Status](https://travis-ci.org/sger/Riseset.svg?branch=master)](https://travis-ci.org/sger/Riseset)

Setup
-----

```ruby
`pod install`
```

Testing
-----

```ruby
`./bin/test`
```
